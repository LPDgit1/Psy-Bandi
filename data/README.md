# Archivio pubblico Streamlit

`bandi.sqlite` viene generato da GitHub Actions e letto da Streamlit in sola lettura.
Il file non è un backup del database applicativo: contiene esclusivamente bandi
approvati, fonti e allegati pubblici, oltre ai metadati di generazione.

Non contiene iscrizioni agli alert, indirizzi email, log di invio, audit
redazionali, esiti tecnici degli import o credenziali.
